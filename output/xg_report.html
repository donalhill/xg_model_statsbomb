About the Data

Data Source

This dashboard uses StatsBomb Open Data, a free dataset of event-level football data released for research and education.

Dataset

9,168 shots (1,014 goals) from all 380 matches of La Liga 2015/16. Each shot includes location, body part, technique, game state, and StatsBomb's own xG.

Train/Test Split

Training: Aug-Dec 2015 (4,162 shots)
Test: Jan-May 2016 (5,006 shots)
Temporal split prevents data leakage.

Model

XGBoost classifier with isotonic calibration, tuned via Optuna (100 trials). Features include distance to goal, shot angle, body part, and game context.

Penalties

Fixed xG of 0.76 from historical rates. In 2015/16 La Liga, there were 97 penalties with 71.1% (±4.6%) conversion.
